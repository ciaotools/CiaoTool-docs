## 🚀 快速开始 | Quick Start

**Solana 发币步骤：**
1. 打开 [ciaotool.io/en-US/token/create](https://ciaotool.io/en-US/token/create)
2. 连接 Phantom 或 Backpack 钱包
3. 填写代币名称、符号、总量、图标
4. 点击一键创建，确认签名
5. 准备约 0.1–0.3 SOL 手续费

**BSC 发币步骤：**
1. 打开 [bsc.ciaotool.io](https://bsc.ciaotool.io)
2. 连接 MetaMask 钱包
3. 填写代币信息
4. 一键部署
